# 四种权限修饰符

java 中四种权限修饰符：

public > protected > (default) > private

```java
/**
 * 四种权限修饰符
 *
 * 所在位置     public > protected > (default) > private
 * 同一个类       yes        yes          yes        yes
 * 同一个包       yes        yes          yes        no
 * 不同包子类     yes        yes          no         no
 * 不同包非子类   yes        no           no         no
 */
```

private: 私有的，只有在同一个类中才能访问。

(default): 默认，同一个包中的所有类都能访问。

protected: 受保护的，同一个包中的所有类都能访问，并且不同包的子类也可以访问。

public: 公共的，都能访问。
